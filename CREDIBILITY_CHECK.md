# 新闻可信度审核机制说明

## 📋 问题背景

即使开启了联网搜索，AI 仍然可能产生事实错误，例如：
- 将虚构新闻当作真实事件分析
- 使用过时的背景信息（如"特朗普2024大选"）
- 未验证新闻中的人物、职位、事件是否真实

## 🎯 解决方案

### **多层防护机制**

```
RSS 抓取
    ↓
深度分析 + 事实核查（新增）
    ↓
   ┌──────────────────────────────┐
   │ credibility = low   → 跳过   │
   │ credibility = medium → 警告  │
   │ credibility = high  → 正常   │
   └──────────────────────────────┘
    ↓
生成点评
    ↓
输出时附带可信度标记
```

---

## 🔧 实现细节

### **1. 时间戳注入**

在 AI 分析时明确告知当前时间：

```python
【重要提示】当前时间是 2026年04月13日。请基于此时间点进行分析，确保所有时间相关的表述准确无误。
```

**效果：**
- 避免使用过时信息
- 确保时间相关表述准确

---

### **2. 前置事实核查**

在生成分析前，AI 会自动核查：

```python
【第一步：事实核查】
在开始分析之前，请先核查以下内容：
1. 文中提到的人物姓名和职位是否真实存在？
2. 文中的核心事件是否有可信来源？
3. 是否存在明显与常识矛盾的内容？
```

**输出格式：**
```json
{
  "credibility": {
    "level": "high/medium/low",
    "issues": ["问题1", "问题2"],
    "notes": "可信度说明"
  }
}
```

---

### **3. 拦截逻辑**

**可信度分级：**

| 等级 | 说明 | 处理方式 |
|------|------|----------|
| **high** | 可信度高 | 正常生成点评 |
| **medium** | 可信度中等 | 生成点评，但添加警告标记 |
| **low** | 可信度低 | 跳过，不生成点评 |

**代码实现：**
```python
# 可信度检查
credibility = analysis.credibility or {}
credibility_level = credibility.get("level", "unknown")
issues = credibility.get("issues", [])

# 如果可信度为 low，跳过该新闻
if credibility_level == "low":
    print(f"  [跳过] 可信度过低: {item['news'].title[:50]}...")
    if issues:
        print(f"    问题: {', '.join(issues[:2])}")
    skipped_count += 1
    continue

# 如果可信度为 medium，添加警告标记
if credibility_level == "medium":
    print(f"    [警告] 可信度中等: {', '.join(issues[:2])}")
```

---

### **4. 可信度标记**

**在生成的 Markdown 中添加警告：**

```markdown
# 新闻标题

> **来源**: 新闻来源 | **时间**: 2026-04-13 12:00:00

> ⚠️ **可信度警告**: 新闻中存在未经验证的内容
> 存在问题: 人物职位可能不真实, 事件缺乏可信来源

---

## 核心要点

...
```

**Front Matter 中也包含可信度：**
```yaml
---
title: "新闻标题"
credibility: medium
---
```

## 💡 注意事项

### **1. 可信度评估不是 100% 准确**

- AI 的判断可能出错
- 建议人工复核重要内容
- 对于关键新闻，建议手动验证

### **2. 联网搜索的局限性**

- DeepSeek 的联网搜索可能只搜到假新闻本身
- 建议后续添加交叉验证搜索
- 可以搜索多个来源验证

### **3. 可信度等级的调整**

可以在代码中调整拦截策略：

```python
# 更严格：跳过 medium 级别
if credibility_level in ["low", "medium"]:
    skipped_count += 1
    continue

# 更宽松：只警告，不跳过
if credibility_level == "low":
    print(f"  [警告] 可信度过低: {item['news'].title[:50]}...")
    # 不跳过，继续生成
```
