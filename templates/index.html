🚀 快速开始

NewsPush 是一个自动化新闻分析和多平台发布系统。


查看新闻 分析新闻 发布文章

📋 系统功能

  • 新闻抓取 - 自动从多个 RSS 源抓取新闻
  • AI 分析 - 深度分析新闻内容和价值
  • 智能点评 - 生成专业的新闻点评文章
  • 多平台发布 - 一键发布到微信公众号、小红书等平台
  • 任务管理 - 查看和管理发布任务

🔧 API 接口

系统提供以下 REST API 接口：

  • GET /api/news/recent - 获取最近新闻
  • POST /api/news/analyze - 分析新闻
  • POST /api/news/commentary - 生成点评
  • POST /api/news/publish - 发布到多平台
  • POST /api/workflow/full - 完整流程（分析+点评+发布）
